Book Placement in a Library Shelf

A librarian is organizing books on a shelf where each book has a unique ID. The book IDs are sorted in ascending order. When a new book arrives, the librarian needs to determine the exact position where it should be placed while maintaining the sorted order.

Given a sorted array of book IDs and a new book's ID (target), find the index where the book should be placed. If the book ID already exists, return its current index.



Input Format

An integer represents the array size
An integer array nums of distinct book IDs sorted in ascending order.
An integer target representing the new book's ID.


Constraints

1 ≤ nums.length ≤ 10^4
-10^4 ≤ nums[i], target ≤ 10^4


Output Format

An integer representing the index where the book should be placed or found.


Example 1

Sample Input 1

4

1 3 5 6

5



Sample Output 1

2



Explanation

Since 5 is already present at index 2, return 2.



Example 2

Sample Input 2

4

1 3 5 6

2



Sample Output 2

1



Explanation

2 is not present in the array, but it should be inserted at index 1 to maintain sorted order.



Example 3

Sample Input 3

4

1 3 5 6

7



Sample Output 3

4



Explanation

7 is larger than all elements, so it should be inserted at the end at index 4.
