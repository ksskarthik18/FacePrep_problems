Jumbled Number Validator

You’ve been assigned to develop a software module for a futuristic security system that requires numeric passcodes. However, this system doesn’t accept just any number — it only allows access when a special type of number, called a Jumbled Number, is entered.

A Jumbled Number is defined as a number in which every pair of adjacent digits has an absolute difference of at most 1. That means, for each digit in the number (except the last), the next digit must be either the same, one less, or one more. If any adjacent digits differ by more than 1, the number is considered invalid and access is denied.

This constraint is designed to make passcodes easy to remember and quick to type using adjacent keys on a keypad, while still offering a unique form of validation.

Your task is to write a program that checks whether a given number satisfies this "jumbled" condition and prints the appropriate result.



Input Format

A single integer n (1 ≤ n ≤ 10⁹) — the number to check.



Output Format

Print "Jumbled" if the number is a jumbled number.
Print "Not Jumbled" otherwise.


Example 1

Sample Input 1

1212



Sample Output 1

Jumbled



Explanation

• Digits: 1 → 2 → 1 → 2

• Differences: |1−2| = 1, |2−1| = 1, |1−2| = 1

• All differences ≤ 1 ⇒ Valid Jumbled Number



Example 2

Sample Input 2

135



Sample Output 2

Not Jumbled



Explanation

Digits: 1 → 3 → 5
Differences: |1−3| = 2 ⇒ Invalid ⇒ Not a Jumbled Number
