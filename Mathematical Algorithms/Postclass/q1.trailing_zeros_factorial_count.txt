Trailing Zeros in Factorial Count
In a mathematical modeling competition, participants are working with large factorials to analyze patterns in number endings. A factorial of a number n (denoted as n!) is the product of all positive integers less than or equal to n. One of the challenges is to determine how many trailing zeroes are present in the factorial of a given number.

Trailing zeroes are created by factors of 10, and since 10 = 2 × 5, and factors of 2 are always more abundant than 5 in a factorial, the number of trailing zeros is determined by the number of times 5 is a factor in the numbers from 1 to n.

Your task is to write a program that calculates the number of trailing zeroes in the factorial of a given integer n.



Input Format

A single integer n (0 ≤ n ≤ 10⁹)



Output Format

Print a single integer — the number of trailing zeroes in n!.


Example 1

Sample Input 1

3



Sample Output 1

0



Explanation

3! = 6 → No trailing zeros.



Example 2

Sample Input 2

5



Sample Output 2

1



Explanation

5! = 120 → One trailing zero.



Example 3

Sample Input 3

0



Sample Output 3

0



Explanation

0! = 1 → No trailing zeros.
