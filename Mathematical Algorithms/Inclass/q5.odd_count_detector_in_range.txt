Odd Count Detector in a Given Range
In a smart traffic signal system, there's a need to estimate the number of odd-numbered cars (based on registration numbers) passing through a segment of road marked with lower and upper number bounds. The system is tasked with identifying how many odd-numbered cars (numbers ending in 1, 3, 5, 7, 9) fall within a given number range [low, high].

As the system has to process data at lightning speed, your algorithm must run in constant O(1) time. Your goal is to help the system return the count of odd numbers between two given non-negative integers low and high, inclusive.



Input Format

The first line contains two integers low and high.



Output Format

Print a single integer — the count of odd numbers between low and high (inclusive).


Example 1

Sample Input 1

3 7



Sample Output 1

3



Explanation

Odd numbers between 3 and 7 are [3, 5, 7]. Total = 3.



Example 2

Sample Input 2

8 10



Sample Output 2

1



Explanation

Odd numbers between 8 and 10 are [9]. Total = 1.
