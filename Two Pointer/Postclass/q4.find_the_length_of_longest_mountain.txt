Find the Length of the Longest Mountain in a Terrain Profile
A team of geologists is analyzing a terrain profile represented by an array of elevation levels. They are particularly interested in identifying "mountain" formations. A mountain is defined as a sequence of at least 3 elements that strictly increases to a peak and then strictly decreases.

Write a program that reads an array of integers representing terrain heights and finds the length of the longest mountain subarray. If there is no such mountain, return 0.



Input Format

First line contains an integer n (1 ≤ n ≤ 10⁴) – number of elements in the array.
Second line contains n space-separated integers representing the array arr (0 ≤ arr[i] ≤ 10⁴).


Output Format

Print a single integer – the length of the longest mountain subarray. If none exists, print 0.


Example 1

Sample Input 1

7

2 1 4 7 3 2 5



Sample Output 1

5



Explanation

The longest mountain is [1, 4, 7, 3, 2]. It increases to 7 and then decreases to 2. The total length is 5.



Example 2

Sample Input 2

3

2 2 2



Sample Output 2

0



Explanation

All elements are equal; there is no increasing or decreasing pattern, hence no valid mountain.
