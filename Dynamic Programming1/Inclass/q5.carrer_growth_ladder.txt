Career Growth Ladder

An employee wants to analyze their career growth by looking at their performance scores over the years.

Each year, the employee receives a performance score (a positive integer). A successful career trajectory would be indicated by a longest increasing subsequence (LIS) of performance scores — that is, a sequence where each next score is higher than the previous one.

Given a list of yearly performance scores, help the employee find the length of the longest increasing subsequence.



Note

A subsequence is a sequence that can be derived by deleting some or no elements without changing the order of the remaining elements.
The numbers must be strictly increasing.
If only 1 score is given for a test case return 0


Input Format

The first line contains an integer n — the number of years.
The second line contains n space-separated integers representing the performance scores.


Constraints

1 <= n <= 1000
1 <= performance score <= 10^4


Output Format

Output a single integer — the length of the longest increasing subsequence.


Example 1

Sample Input 1

6

5 1 6 2 7 3



Sample Output 1

3



Explanation

One of the longest increasing subsequences is [5, 6, 7] or [1, 2, 3], each of length 3.



Example 2

Sample Input 2

8

10 22 9 33 21 50 41 60



Sample Output 2

5



Explanation

One of the longest increasing subsequences is [10, 22, 33, 50, 60], which has a length of 5.
