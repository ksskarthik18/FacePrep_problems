Vending Machine: Total Ways to Dispense Change
A vending machine is stocked with an infinite supply of different coin types.

Given a list of coin denominations and a target amount, your task is to count the total number of combinations that the machine can use to make up the target amount.

You can use any number of coins of each type.



Input Format

The first line contains two integers n and sum — the number of different coin denominations and the target sum.
The second line contains n space-separated integers representing the coin denominations.


Constraints

1 <= n <= 100
1 <= sum <= 10^4
1 <= coin value <= 10^4


Output Format

Output a single integer — the total number of ways to make up the target sum.


Example 1

Sample Input 1

3 4

1 2 3



Sample Output 1

4



Explanation

The 4 ways are:

1 + 1 + 1 + 1
1 + 1 + 2
2 + 2
1 + 3


Example 2

Sample Input 2

2 5

1 2



Sample Output 2

3



Explanation

The 3 ways are:

1 + 1 + 1 + 1 + 1
1 + 1 + 1 + 2
1 + 2 + 2
