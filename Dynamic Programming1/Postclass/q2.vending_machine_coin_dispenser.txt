Vending Machine Coin Dispenser
A vending machine needs to return change to customers using the fewest number of coins possible.

You are given an array of coin denominations and a total amount of money. Write a program to find the minimum number of coins needed to make up the given amount.

If it is not possible to make up that amount with the given coins, return -1.



Input Format

The first line contains two integers n and amount — the number of different coin denominations and the target amount respectively.
The second line contains n space-separated integers representing the coin denominations.


Constraints

1 <= n <= 12
0 <= amount <= 10^4
1 <= coin value <= 10^4


Output Format

Output a single integer — the minimum number of coins needed, or -1 if it is not possible.


Example 1

Sample Input 1

3 11

1 2 5



Sample Output 1

3



Explanation

The machine can use coins 5 + 5 + 1 to make 11, which uses 3 coins.



Example 2

Sample Input 2

2 3

2 4



Sample Output 2

-1



Explanation

It is not possible to make 3 using coins 2 and 4.
